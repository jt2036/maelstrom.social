# Contributing

Thanks for helping build `maelstrom.social`.

## Dev Setup

### Prereqs
- Node.js `>=18`
- npm (comes with Node)

### Run the web app
```bash
cd maelstrom
npm install
npm run dev
```

Then open `http://localhost:3000`.

### Useful scripts
```bash
cd maelstrom
npm run lint
npm run type-check
npm run build
npm run start
```

## Guidelines
- Keep changes small and focused.
- Prefer plain, readable TypeScript over clever abstractions.
